Get Transactions

Get a list of all transactions​

GET https://kernelserver.{env}.haloplus.io/transactions

Path Parameters​

NameTypeDescription
env*StringThe backend environment [dev, qa, prod]

Query Parameters​

NameTypeDescription
?integrator=:id*StringIf you are an integrator, your integrator id is required. Returns all transactions belonging to an integrator.
?merchant=:id*StringIf you a merchant, your merchant id is required. Returns all transactions belonging to an merchant.
?user=:id*StringIf you want transactions limited to a user, add the user id. Returns all transactions belonging to the user.
?startDate=YYYY-MM-DDStringFilters transactions to those created on or after this date.
?endDate=YYYY-MM-DDStringFilters transactions to those created on or before this date.
?limit=:numberNumberMaximum number of results to return.
?offset=:numberNumberNumber of records to skip for pagination.
?minAmount=:numberNumberFilters transactions with amounts greater than or equal to this value.
?maxAmount=:numberNumberFilters transactions with amounts less than or equal to this value.
?merchantReference=:stringStringFilters transactions by the provided merchant reference.
?transactionDisposition=:valueNumberFilters by transaction disposition status. Accepted values:
  • Approved
  • Declined
  • DeclinedOffline
  • UnableToGoOnline
  • Indeterminate
  • Voided

NameTypeDescription
x-api-key*StringThe API Key retrieved from the Merchant Portal
