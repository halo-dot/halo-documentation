# Get Transactions

## Get a list of all transactions

<mark style={{color:'blue'}}>`GET`</mark> `https://kernelserver.{env}.haloplus.io/transactions`

#### Path Parameters

| Name                                      | Type   | Description                              |
| ----------------------------------------- | ------ | ---------------------------------------- |
| env<mark style={{color:'red'}}>\*</mark>     | String | The backend environment \[dev, qa, prod] |

#### Query Parameters

| Name                                                   | Type             | Description                                                                                                                                                                                                  |
| ------------------------------------------------------ | ---------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| ?integrator=:id<mark style={{color:'red'}}>\*</mark> | String           | If you are an integrator, your integrator id is required. Returns all transactions belonging to an integrator.                                                                                               |
| ?merchant=:id<mark style={{color:'red'}}>\*</mark>   | String           | If you a merchant, your merchant id is required. Returns all transactions belonging to an merchant.                                                                                                          |
| ?user=:id<mark style={{color:'red'}}>\*</mark>       | String           | If you want transactions limited to a user, add the user id. Returns all transactions belonging to the user.                                                                                                 |
| ?startDate=YYYY-MM-DD                                  | String           | Filters transactions to those created on or after this date.                                                                                                                                                 |
| ?endDate=YYYY-MM-DD                                    | String           | Filters transactions to those created on or before this date.                                                                                                                                                |
| ?limit=:number                                        | Number           | Maximum number of results to return.                                                                                                                                                                         |
| ?offset=:number                                       | Number           | Number of records to skip for pagination.                                                                                                                                                                    |
| ?minAmount=:number                                    | Number           | Filters transactions with amounts greater than or equal to this value.                                                                                                                                       |
| ?maxAmount=:number                                    | Number           | Filters transactions with amounts less than or equal to this value.                                                                                                                                          |
| ?merchantReference=:string                            | String           | Filters transactions by the provided merchant reference.                                                                                                                                                     |
| ?transactionDisposition=:value                        | Number           | Filters by transaction disposition status. Accepted values:<ul><li>`Approved`</li><li>`Declined`</li><li>`DeclinedOffline`</li><li>`UnableToGoOnline`</li><li>`Indeterminate`</li><li>`Voided`</li></ul> |

#### Headers

| Name                                        | Type   | Description                                    |
| ------------------------------------------- | ------ | ---------------------------------------------- |
| x-api-key<mark style={{color:'red'}}>\*</mark> | String | The API Key retrieved from the Merchant Portal |
